About The Website

This "50xGallery" website is my cs50x final project, and it is all my individual work 50xGallery is a simple and friendly drawing website that allows you to create digital art directly in your browser. The main idea behind this project is to make drawing easy and accessible for everyone.

On this website, you can:

  • You can Register, Login and Logout
  • Draw freely using the Pen tool
  • Use the Eraser to fix your mistakes
  • Choose from multiple colors
  • Choose from a color picker
  • Change the pen size
  • Save your artwork directly to your computer

The goal of this project is to offer a clean, simple, and enjoyable drawing experience for all users.
